# SQL_Excel_Dashboard_PizzaSales
Interactive dashboard in excel about Pizaa sales.
Used SQL for veryfing the KPI findings and helping me learn more about querying.

 Project Overview
This project demonstrates end-to-end data analysis and visualization using PostgreSQL and Excel. The dataset contains detailed pizza sales transactions, and the goal was to show interactive dashboard that highlights key business KPIs and actionable insights.

The dashboard integrates:

SQL queries for data extraction and KPI calculation

Excel PivotTables & Charts for visualization

Interactive filters & slicers for dynamic exploration
